<h2 align="center" style="color: #FF6347;">✨ Key Features ✨</h2>

<div align="center">
  <ul style="text-align: left; max-width: 700px; font-size: 1.1em; line-height: 1.6; background-color: #ffffff; padding: 20px; border-radius: 15px; box-shadow: 0 5px 15px rgba(0,0,0,0.1); margin: 20px 0;">
    <li>🔍 <strong>Advanced Document Retrieval</strong>: Utilizes custom encoders and Neo4j for efficient document storage and retrieval.</li>
    <li>⚡ <strong>High-Performance Caching</strong>: Implements optimized caching strategies for faster response times.</li>
    <li>📰 <strong>Background Scraping Service</strong>: Automatically scrapes news articles in a separate thread to keep the data up-to-date.</li>
    <li>📑 <strong>Comprehensive API Endpoints</strong>: Provides endpoints for health checks, search functionality, and user management.</li>
    <li>🔒 <strong>User Request Management</strong>: Monitors user request frequency and manages rate-limiting.</li>
    <li>🐳 <strong>Dockerized Deployment</strong>: Easily deploy the application using Docker for consistent and scalable environments.</li>
    <li>📊 <strong>API Logging and Inference Time Recording</strong>: Tracks API performance and request details for better observability.</li>
  </ul>
